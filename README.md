# Press Kit — kito (@eukitoo)

Landing page profissional inspirada na estrutura do [Press Kit PRO](https://presskitpro.app/rodrigodantas/), adaptada para artista de streaming (não DJ).

## Como usar

1. Instale e rode o servidor local:

```bash
npm install
npm run dev
```

Abre em **http://localhost:5173** (YouTube e fotos funcionam melhor que abrir o HTML direto).

2. Edite **`config.js`** com links reais (Spotify, Apple, etc.).
3. Fotos em **`assets/`** (`kito-01.jpg` …) — lista em `config.js` → `gallery`.
